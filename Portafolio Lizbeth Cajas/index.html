GENERADOR DE HORARIO

Pagina web generardor de horario para docentes

Diseño de una pagina web con un sistema de generador de horarios y visualizacion para los docentes con la conexion de base de datos sql, la creacion de los usuarios para que puedan acceder a través de roles definidos, como administrador, docente y laboratorista, cada uno con sus propias funcionalidades y permisos específicos

El generador de horario para docentes es una aplicación web diseñada para simplificar y optimizar la gestión de horarios en instituciones educativas. Su objetivo principal es facilitar la asignación de clases y actividades, teniendo en cuenta los horarios disponibles de los docentes, la disponibilidad de aulas y recursos, y las preferencias individuales.

Sistema de egresos e ingresos para la familia

El Sistema de Egresos e Ingresos para la Familia es una página web diseñada para facilitar la gestión financiera del hogar. Esta plataforma online permite a las familias registrar y monitorear sus ingresos y gastos de manera eficiente

La página web calcula automáticamente los ingresos totales en un período específico, brindando a los usuarios una visión clara de sus recursos financieros disponibles, se incluye una función de generación de gráficos que muestra visualmente los ingresos y gastos a lo largo del tiempo. Los usuarios pueden elegir entre diferentes tipos de gráficos para analizar sus finanzas de manera más efectiva.

Sistema de facturacion

Sistema de facturacion de una farmacia

El Sistema de Facturación para una farmacia es una aplicación Java que utiliza una base de datos MongoDB para almacenar y gestionar la información relacionada con productos, inventario, ventas y clientes.

El sistema permite llevar un registro actualizado del inventario de la farmacia, incluyendo información detallada de los productos disponibles, como nombre, descripción, precio, cantidad en stock y fecha de vencimiento.
Los empleados pueden registrar las ventas de productos de manera rápida y sencilla a través de la aplicación. Cada venta se registra en la base de datos, incluyendo los detalles de los productos vendidos, el precio unitario, la cantidad y el total de la venta.
